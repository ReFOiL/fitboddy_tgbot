1. Общее описание проекта
1.1. Назначение системы
Fitboddy — это интеллектуальный Telegram-бот персонального фитнес-тренера. Система автоматизирует процесс создания индивидуальных тренировочных программ на основе анкетных данных пользователя, предоставляет медиаконтент по технике выполнения упражнений и инструменты для отслеживания питания.

1.2. Цели проекта
Автоматизация работы фитнес-тренера

3.3. Модуль питания (КБЖУ)
3.3.1. Калькулятор норм (бесплатный тариф)
Формула расчета: Миффлин-Сан Жеора с модификациями

python
BMR = (10 * weight) + (6.25 * height) - (5 * age) + 5 (для мужчин)
BMR = (10 * weight) + (6.25 * height) - (5 * age) - 161 (для женщин)

TDEE = BMR * коэффициент активности:
- sedentary: 1.2
- light: 1.375
- moderate: 1.55
- active: 1.725
- athlete: 1.9

Целевые калории = TDEE * коэффициент цели:
- weight_loss: 0.8
- muscle_gain: 1.2
- maintenance: 1.0

Распределение БЖУ:
- Белки: 1.6-2.2 г/кг веса
- Жиры: 0.8-1.2 г/кг веса
- Углеводы: остаток калорий
3.3.2. Дневник питания (платный тариф)
